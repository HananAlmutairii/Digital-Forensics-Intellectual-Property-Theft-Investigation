# Digital-Forensics-Intellectual-Property-Theft-Investigation

Digital forensics investigation involving PCAP analysis, registry artefacts, EXIF metadata correlation, and evidence reconstruction in a simulated intellectual property theft case.

## Investigation Analysis

[View Investigation Analysis](https://hananalmutairii.github.io/Digital-Forensics-Intellectual-Property-Theft-Investigation/)

A digital forensics investigation focused on reconstructing user activity, analysing network traffic, correlating registry artefacts, and identifying indicators of data exfiltration in a simulated intellectual property theft case.

## Investigation Capabilities Demonstrated

* File system analysis
* TCP stream reconstruction
* Registry artefact analysis
* Timeline reconstruction
* Deleted file recovery
* EXIF metadata correlation
* Obfuscation detection
* Multi-source evidence correlation

## Tools Used

* Autopsy
* Wireshark
* Registry Explorer
* FTK Imager
* The Sleuth Kit

## Notes

This project is based on an academic digital forensics investigation scenario. Full forensic images, evidence files, and coursework solutions are not publicly shared.
